DATABASE SETUP

SQL Server & Entity Framework Core 10

← Volver al Índice Técnico

1. Requisitos del Servidor

Limen utiliza SQL Server 2019+ (instancia local o remota).

2. Connection String

Configure la cadena de conexión en appsettings.json:

"ConnectionStrings": {
  "DefaultConnection": "Server=db.example.com;Database=LimenDB;User Id=sa_user;Password=FakePassword123!;TrustServerCertificate=True;"
}

3. Aplicación de Migraciones

Ejecute la herramienta DbMigrator para crear el esquema y los datos iniciales (Seed):

dotnet run --project tools/Limen.DbMigrator

4. Datos de Semilla (Seed)

Al finalizar la migración, se crean automáticamente los siguientes datos de acceso inicial:

  • Tenant: demo
  • Admin: admin@limen.app / Password: Limen!App2026
Seguridad: Se recomienda encarecidamente cambiar la contraseña del usuario administrador inmediatamente después del primer inicio de sesión.
