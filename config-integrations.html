System Integrations

Conexión con Entra ID, RRHH y APIs de Terceros

← Volver al Índice Técnico

1. Identidad (Entra ID / AD)

Configure el App Registration en Azure Portal y proporcione el ClientId y TenantId en Limen.

2. RRHH (GeoVictoria)

La integración con GeoVictoria permite sincronizar bajas laborales en tiempo real. Se requiere el endpoint de la API y el Token de autorización.

3. Webhooks de LPR

Las cámaras LPR deben apuntar a: https://api.limen.app/api/lpr/event incluyendo el header X-Api-Key.
