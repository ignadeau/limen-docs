1. Docker Compose

El despliegue estándar utiliza 3 servicios:

  • limen-web: Portal Blazor (Puerto 8080).
  • limen-api: API REST (Puerto 8081).
  • limen-mobile-api: API Móvil (Puerto 8082).

2. Nginx

Se recomienda usar Nginx como reverse proxy para gestionar SSL y subdominios dinámicos.
