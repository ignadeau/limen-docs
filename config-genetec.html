1. Web SDK y Web Server

La integración requiere que el servidor de Genetec tenga instalado el rol de Web SDK. Limen utiliza REST API para interactuar con el sistema.

2. Autenticación y Seguridad

Limen utiliza autenticación de tipo "OAuth2" o "Basic" según la configuración de Genetec. Se recomienda el uso de un certificado SSL válido para evitar fallos de seguridad.

3. Orquestación de Synergis (Acceso)

Limen gestiona los Cardholders y sus Credentials:

Flujo Maestro: Cuando una visita se auto-registra en Limen, se crea un Temporary Cardholder en Genetec y se le asigna un Access Rule específico por el tiempo que dure su invitación.

4. Integración LPR (AutoVu)

Limen consume los eventos de lectura de patentes desde el motor AutoVu, permitiendo validar aforo vehicular en complejos industriales de alta densidad.
