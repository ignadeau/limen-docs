ERP INTEGRATION (SAP / WORKDAY)

Sincronización de Estructura Organizacional y Jerarquías

← Volver al Índice Técnico

1. Sincronización Organizacional

Para grandes corporaciones, Limen hereda la estructura de departamentos, jefaturas y centros de costo directamente desde el ERP (SAP SuccessFactors o Workday).

2. Integración SAP SuccessFactors

Limen utiliza el protocolo OData v2/v4 para leer las entidades User y EmpJob.

  • Autenticación: OAuth 2.0 con Certificado X.509.

3. Integración Workday

Mediante el servicio Workday Web Services (WWS), Limen consulta los eventos de ciclo de vida (Hire, Terminate, Transfer).
