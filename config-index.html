Guía Maestra de Configuración

Siga estos manuales en orden secuencial para desplegar una instancia completa de Limen.

🚀 Simulador de Hardware

Pruebe todos los flujos del sistema (altas, bajas, aperturas) sin necesidad de equipos físicos.

1. Base de Datos

SQL Server 2019+, Migraciones EF Core y Seed Data.

2. Almacenamiento (Storage)

Configuración de S3, Azure Blob y cifrado DataProtection.

3. Sincronización de Identidad

Automatización de altas/bajas con Entra ID, AD y sistemas de RRHH.

4. Manual Maestro ZKTEco

La guía definitiva para la integración total: Hardware, Software y API.

5. Integración Hikvision

Gestión de HikCentral, terminales MinMoe y cámaras LPR.

6. Integración Genetec

Conectividad Enterprise con Synergis, Omnicast y AutoVu.

7. Integración de Video (VMS)

Vinculación de cámaras, clips de evidencia y búsqueda unificada.

8. Integración GeoVictoria

Sincronización de asistencia y dotación mediante Polling (Consulta periódica).

9. Integración Buk

Sincronización en tiempo real mediante Webhooks y seguridad HMAC-SHA256.

9b. Integración SAP SuccessFactors

Employee Central via OData v2 y OAuth 2.0 Client Credentials — para empresas con ecosistema SAP.

10. Notificaciones WhatsApp

Envío de QRs y alertas de seguridad vía Twilio / WhatsApp Business API.

11. Centrales de Incendio

Liberación automática de torniquetes y Muster List.

12. ERP (SAP / Workday)

Gestión de estructura organizacional y transferencias.

13. Despliegue (Docker)

Orquestación con Docker Compose y Nginx Proxy.
