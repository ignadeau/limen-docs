1. Gestión de Clientes (Tenants)

El System Admin es el único con capacidad para crear y suspender clientes (Tenants) en la plataforma global.

Campo Requerido Descripción
Nombre del Tenant * SÍ Nombre legal de la empresa cliente.
Subdominio * SÍ Identificador único para la URL (ej: cliente para cliente.limen.app).
Email de Contacto * SÍ Correo del administrador principal del cliente.

2. Configuración Global de Almacenamiento

Define los backends disponibles (Local, S3, Azure) para que los Tenants los configuren.

Tarea Proactiva: Monitorear los Audit Logs globales para detectar intentos de acceso fallidos o cambios en la configuración de seguridad.
