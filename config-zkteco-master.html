Paso 1 — Generar la API Key en ZKBio CVSecurity

La API Key es un token estático que autoriza a Limen a operar contra la API REST de ZKBio. Se genera UNA sola vez y se copia en Limen.

  1. Inicia sesión en el panel web de ZKBio CVSecurity con cuenta de administrador.
  2. En el menú lateral, ve a System → API Management (en algunas versiones: Authorization Management → API Management).
  3. Haz clic en Add o Generate Token.
  4. Completa el formulario:
    CampoValor sugerido
    Name / DescriptionLimen-Integration
    PermissionsSeleccionar: Access Control, Transaction Monitor, Person Management
    ExpiryNever (o la política de tu organización)
  5. Haz clic en Save / Confirm.
  6. El sistema mostrará el token generado. Cópialo inmediatamente — en algunos servidores ZKBio el token completo no vuelve a mostrarse después de cerrar la ventana.
Ejemplo de token: eyJhbGciOiJIUzI1NiJ9.eyJzdWIiOiIxIiwiY3JlYXRlZCI6MTY4MDAwMDAwMCwiZXhwIjowfQ.XXXXXX

Paso 2 — Configurar la Integración en Limen

  1. Inicia sesión en Limen con rol TenantAdmin.
  2. Ve a Administración → Integraciones.
  3. En la sección Control de Acceso Físico, selecciona el proveedor ZKBio CVSecurity.
  4. Completa los campos que aparecen:
    CampoDescripciónEjemplo
    URL Base Dirección raíz del servidor ZKBio, sin barra final ni rutas adicionales http://192.168.1.100
    https://zkbio.empresa.com
    API Key El token que copiaste en el Paso 1. Se almacena cifrado con DataProtection. eyJhbGci...
  5. Haz clic en Probar conexión. Si todo está bien, el badge cambiará a Activo.
  6. Haz clic en Guardar. Las credenciales quedan cifradas en la base de datos. El campo API Key se limpia de la pantalla por seguridad — eso es normal.
🔒 Seguridad: La API Key se cifra con Microsoft DataProtection antes de guardarse en la base de datos. Nunca aparece en texto plano en logs ni en respuestas HTTP. Si necesitas cambiarla, simplemente ingresa el nuevo valor y guarda de nuevo.

Paso 3 — Verificar la Conexión

Una vez guardada la configuración, puedes verificar que Limen se conecta correctamente al hardware:

  1. Ve a Administración → Infraestructura → Puertas.
  2. Haz clic en Nueva Puerta.
  3. En la sección Importar desde hardware, haz clic en Listar puertas del hardware.
  4. Si la conexión es correcta, aparecerá la lista de puertas configuradas en ZKBio (con su ID y nombre). Si aparece "No se encontraron puertas", revisa la sección Troubleshooting.
  5. Haz clic en Usar este ID junto a la puerta que quieres mapear en Limen. El campo "ID en hardware" se completa automáticamente.
