AD, Entra ID y Sistemas de RRHH (GeoVictoria)

← Volver al Índice Técnico

1. El Ciclo de Vida de la Identidad

Limen automatiza el aprovisionamiento. Cuando RRHH crea un empleado, Limen lo detecta, crea su usuario y le otorga acceso físico automáticamente.

2. Configuración de Entra ID (Azure AD)

Utilizamos el Microsoft Graph SDK para la sincronización.

ParámetroValor Requerido
Application IDGUID de la App en Azure.
Directory IDGUID del Tenant de Microsoft.
Client SecretLlave secreta para acceso offline.
Group FilterFiltro para sincronizar solo grupos específicos (ej: "Planta_Principal").

3. Sincronización de RRHH (GeoVictoria)

Limen consulta el API de GeoVictoria para obtener el estado laboral actual. Si un empleado es marcado como "Finiquitado", Limen dispara la **Regla de los 30 Segundos**.
