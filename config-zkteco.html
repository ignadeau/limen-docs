2. Mapeo de Puertas

El HardwareID en la configuración de puertas de Limen debe coincidir exactamente con el DoorID reportado por el servidor ZK BioSecurity.

3. Troubleshooting

  • Error 401: Credenciales de API incorrectas o sesión expirada.
  • Error 504: Servidor ZK fuera de línea o firewall bloqueando el puerto.
