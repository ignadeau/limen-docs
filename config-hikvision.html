1. Requisitos de la Plataforma

Limen se integra con HikCentral Professional v2.0+ utilizando el protocolo HTTPS y el módulo de OpenAPI.

2. Configuración de Credenciales

En HikCentral, cree un "App" en el centro de API y obtenga:

ParámetroDescripción
AppKeyIdentificador único de la aplicación Limen.
AppSecretFirma para el cifrado de las peticiones (HmacSHA256).
Puerto OpenAPIDefault: 80 para HTTP o 443 para HTTPS.

3. Mapeo de Dispositivos

  • Controladoras de Acceso: Utilice el IndexCode de la puerta en HikCentral como el HardwareID en Limen.
  • Cámaras LPR: Limen recibe el evento de patente vía /api/resource/v1/vehicle/alarm.

4. Sincronización de Biometría Facial

Limen envía las selfies de las visitas directamente a la base de datos de rostros de HikCentral. Al ser aprobado en Limen, el rostro se propaga a los terminales de la serie MinMoe en menos de 10 segundos.
